JWT Authentication
  • JSON Web Tokens are an open, industry standard Rfc 7519 method for representing claims securely between two parties
  • JWT.IO allows you to decode , verify and generate Jwt
  • JWTs can be signed using a secret (with the IMAC algorithm) or a public/private key pair using RSA or ECOSA
  • JWTs signed can verify the integrity of the claims contained within it
